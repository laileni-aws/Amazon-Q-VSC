# Amazon Q for VS Code - Product Overview

## Purpose

Amazon Q for VS Code is a generative AI-powered assistant extension that accelerates software development by providing intelligent code suggestions, chat-based assistance, security scanning, and code transformation capabilities directly within Visual Studio Code.

## Value Proposition

-   Increases developer productivity through AI-powered inline code completions
-   Provides conversational AI assistance for code generation, explanation, and debugging
-   Automates security vulnerability detection and remediation
-   Enables Java application modernization and upgrades
-   Supports both free tier (AWS Builder ID) and Pro tier (SSO) authentication

## Key Features

### Agentic Coding

-   Natural language instruction processing for code generation
-   File reading, code diff generation, and command execution
-   Iterative feedback-based code modifications
-   MCP (Model Context Protocol) server integration for extended context

### Inline Code Suggestions

-   Real-time code completions as you type
-   Support for 15+ languages (Python, TypeScript, Rust, Terraform, CloudFormation, etc.)
-   Reference tracking for licensed code suggestions

### Chat Interface

-   Code generation from natural language prompts
-   Code explanation and documentation
-   Software development Q&A
-   Context-aware responses using workspace files

### Security Scanning

-   Automated vulnerability detection (SAST)
-   Security issue tree view with severity indicators
-   AI-powered fix generation and explanation
-   Issue filtering and grouping capabilities

### Code Transformation (Q Transform)

-   Java application upgrades and modernization
-   Transformation plan visualization
-   Proposed changes review workflow
-   Transformation hub for job management

## Target Users

-   Individual developers using AWS Builder ID (Free Tier)
-   Enterprise teams with Amazon Q Developer Pro subscriptions
-   Developers working with AWS services and infrastructure
-   Teams requiring AI-assisted code review and security scanning

## Use Cases

-   Implementing new features with AI assistance
-   Generating unit tests and documentation
-   Performing automated code reviews
-   Upgrading legacy Java applications
-   Analyzing and fixing security vulnerabilities
-   Learning new APIs and frameworks through chat
